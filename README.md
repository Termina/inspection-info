## Inspect Info

> a bunch of commands for dev informations.

### Usages

Install it with cargo install git source.

##### IP address

```bash
in ip
10.143.35.141
```

```bash
in ip -d
lo0:	127.0.0.1
lo0:	::1
lo0:	fe80::1
en0:	192.168.35.141
```
